# 11s 的 C&Cpp 笔记本

这是一个记录 C++ 学习过程的仓库，包含每周的练习代码和笔记。

## 第七周练习 (Week7)

### basic_io.cc
基本输入输出练习，包括：
+ 单个数据读取（C++和C风格）
+ 多组数据读取
+ 单行多个数据读取（空格和逗号分隔）

### problem_1_a_plus_b.cc
计算两个整数的和，包括：
+ C风格和C++风格的实现
+ 多组数据处理示例

### problem_2_formula_solve.cc
一元二次方程求解，包括：
+ 处理各种特殊情况（a=0, b=0, 判别式小于0等）
+ 使用C语言实现，处理逗号分隔的输入

### problem_3_bubble_sort.cc
冒泡排序实现，包括：
+ 排序函数
+ 对4个整数进行排序
+ 多组数据处理

### problem_4_triangle.cc
打印三角形图案，包括：
+ 三角形打印函数
+ 打印倒三角形
+ 多组数据处理

### problem_5_ppdi.cc
水仙花数判断，包括：
+ 水仙花数判断函数
+ 查找指定范围内的所有水仙花数
+ 结果格式化输出

## 第八周练习 (Week8)

### problem_1_is_prime.cc
素数判断，包括：
+ 素数判断函数
+ 优化的素数判断算法（只检查到平方根，跳过偶数）
+ 多组数据处理

### problem_2_gcd.cc
最大公约数计算，包括：
+ 最大公约数函数
+ 递归实现的辗转相除法
+ 简洁的现代写法

### problem_3_nine_nine_table.cc
九九乘法表打印，使用嵌套循环实现标准的九九乘法表

### problem_4_frac_to_dec.cc
分数转小数，包括：
+ 计算整数部分和小数部分
+ 四舍五入处理
+ 处理小数部分为0的情况

### problem_5.cc
进制转换，包括：
+ 进制转换函数
+ 递归实现将十进制数转换为指定进制
+ 处理输入为0的特殊情况
